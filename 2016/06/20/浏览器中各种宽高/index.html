浏览器中各种宽高

发表于 2016-06-20 | 分类于 css

原生js取宽高

window document 区别
window对象表示浏览器中打开的窗口(包括菜单栏)
window可以省略,window.console.log(1)等价于console.log(1)
document对象是浏览器的html文档
是window的一部分，即window除去菜单栏剩下的部分
window.document.body 等价于 document.body
window.location===document.location===window.document.location

等价的，无差别

window.innerHeight window.outerHeight

前者指的是即document对象的高度,不包括控制台和浏览器工具栏

后者指的是整个浏览器窗口的高度，包括工具栏等

window.screen表示用户的屏幕

screen.height 整个屏幕高度

screen.availHeight 屏幕高度除去桌面导航栏 即屏幕可利用高度

screenTop 浏览器距离屏幕顶部的距离 与screenX等价

clientWidth clientHeight （document.body.clientWidth）

表示元素可视部分宽度和高度，即padding＋content（content指元素的height,width）

有滚动条时候需要注意，例如某个div长宽200px，overflow:auto
在里面添加字直到滚动条出现时，clientWidth要减去滚动轴宽度，mac下和windows下不同

mac下会对滚动轴进行隐藏处理，windows不会

总结：

无padding无滚动：clientWidth = style.width

有padding无滚动：clientWidth = style.width+style.padding(左右)

有pading有滚动且滚动条显示：clientWidth = style.width+style.padding(左右)－滚动条宽度

clientLeft clientTop

前者表示属性属性的border-left的宽度

document.body.offsetWidth/offsetHeight

offsetWidth表示border+padding+content

该属性和内部的内容是否超出元素的大小无关，只和本来设定的数据有关

小结：
  • 无padding无滚动无border
    offsetWidth=style.width=clientWidth
  • 有padding无滚动有border
    offsetWidth=clientWidth+border
offsetWidth=style.width+padding+boder
  • 有padding有滚动有border
    offsetWidth=clientWidth+滚动+border
offsetWidth=style.width+padding+border
document.body.offsetLeft/offsetTop

与offsetParent有关

当前元素的父级元素没有进行css定位(position:absolute/relative),则offsetParsent为body

当前元素的父级元素有css定位，则offsetParsent为父级元素

ie6/7

offsetLeft = offsetParent的padding+当前元素的margin

ie8/9/10 chrome

offsetLeft = offsetParent的margin、border、padding+当前
元素的margin

firefox

offsetLeft = offsetParent的padding、margin+当前元素的margin

document.body.scrollWidth/scrollHeight

在body中

给定宽度小于浏览器窗口时，scrollWidth通常是浏览器窗口的宽度

给定宽度大于浏览器窗口时，且内容小于给定宽，scrollWidth为其给定的宽度加上padding、border、marign

给定宽度大于浏览器窗口时，且内容大于给定宽，scrollWidth为其内容的宽度加上padding、border、marign

在元素中

无滚动轴时
scrollWidth=clientWidth=style.width+style.padding

有滚动轴时
scrollWidth=实际内容的宽度+padding

scrollTop scrollLeft 可读写的

表示元素卷起的宽度高度

document //document对象
document.documentElement //html
document.body   //body
他们对应的是祖辈父辈子悲的关系

clientX clientY 相对于浏览器可视区左上角的距离
screenX clientY 相对于设备屏幕左上角的距离
offsetX offsetY 相对于事件源左上角的距离（例如某个元素触发的事件）
pageX   pageY   相对于整个网页左上角的距离 （例如在滚动页面时，相对于整个文档的坐标）
X       Y       相对于用css动态定位的最内层包容元素

offset()
相对于document的当前坐标值（相对于body左上角）
不包括该元素的margin padding
position()
相对于offset()的坐标
元素所占有的位置相对于目标的坐标（占有的位置包括margin padding）
父级元素有设置position则 结果为父级元素的padding
若本身也设置了position则结果为 本身的父级的padding加上本身的偏移
