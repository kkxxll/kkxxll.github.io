5种方式定义line-height

发表于 2016-05-21 | 分类于 css 
{line-height:normal;}
{line-height:inherit;}从父元素继承 line-height 属性的值
{line-height:120%;}使用一个百分比
{line-height:25px;}使用一个长度值(px,em等)
{line-height:1.2}使用纯数字

关于line-height的缩写

line-height的值紧跟着font-size值使用斜杠分开，
如：<font-size>/<line-height>

实例：

{font:100%/normal arial;}
{font:100%/120% arial;}
{font:100%/25px arial;}
{font:100%/1.2 arial;}

百分比：

body{
font-size: 15px;
line-height: 200%;
}
h1{
font-size: 20px;
}
line-height为15*2=30px，所有继承的元素都会使用这个值，不会随着相关的font-size做相应的比例缩放

长度：

body{
font-size: 15px;
line-height: 50px;
}
h1{
font-size: 20px;
}
line-height为50px，所有继承的元素都会使用这个值，不会随着相关的font-size做相应的比例缩放

normal：（约为1.2）

body{
font-size: 15px;
line-height:normal;
}
h1{
font-size: 20px;//行高为20*1.2
}
line-height为font-size属性的值乘1.2，所有继承的元素都会使用这个值，会随着相关的font-size做相应的比例缩放

纯数字：

body{
font-size: 15px;
line-height:2;
}
h1{
font-size: 20px;//行高为20*2
}
line-height为font-size属性的值乘2，所有继承的元素都会使用这个值，会随着相关的font-size做相应的比例缩放
