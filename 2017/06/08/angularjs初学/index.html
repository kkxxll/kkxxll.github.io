angularjs初学

发表于 2017-06-08 | 分类于 angularjs

###开始!!：

<html lang="en" ng-app="app">

告诉浏览器开始启动angular,以下例子若没有特殊声明,则默认包含这段代码

控制器：

var app = angular.module("app", []);
app.controller("A", ['$scope', function($scope) {
  $scope.greeting = {
    text: 'hello kk'
  }
}])

<div ng-controller="A">
  <p>{{greeting.text}}</p>
</div>
控制器(Controller)的实现
可以创建多个控制器，若控制器之间有公用的部分，则可以抽出来成服务Service

以下是通过功用控制器实现,servive后面介绍
<div ng-controller="CommonController">
  <div ng-controller="Controller1">
    <p>{{greeting.text}}</p>
    <button ng-click="test1()">test1</button>
  </div>
  <div ng-controller="Controller2">
    <p>{{greeting.text}}</p>
    <button ng-click="test2()">test2</button>
  </div>
  <button ng-click="commonFn()">commonFn</button>
</div>

var app = angular.module("app", []);
  app.controller("CommonController", ['$scope', function($scope) {
    $scope.commonFn = function() {
      alert("通用的");
    };
  }])
  app.controller("Controller1", ['$scope', function($scope) {
    $scope.greeting = {
      text: 'hello1'
    };
    $scope.test1 = function() {
      alert("test1");
    };
  }])
  app.controller("Controller2", ['$scope', function($scope) {
    $scope.greeting = {
      text: 'hello2'
    };
    $scope.test2 = function() {
      alert("test2");
    };
}])
Controller使用注意:
  • 不要复用控制器，一个控制器对应一块视图
  • 不要在控制器里操作dom 这不是控制器的职责
  • 不要在控制器里做数据格式化，ng有自带的表单控件
  • 不要在控制器里做数据过滤，ng有$filter服务
  • 控制器是不会互相调用的，它们之间的交互通过事件进行
标签kk 会被替换成template的内容

Model(数据模型):

双向绑定
不需要自定义js代码，只要引入angular.js，html添加ng-app即可
<!DOCTYPE html>
  <html lang="en" ng-app>
  <head>
    <meta charset="UTF-8">
    <title></title>
  </head>
  <script type="text/javascript" src="angular.js"></script>
  <body>
    <div>
      <input ng-model="say"></input>
      <p>for-model:{{say}}</p>
    </div>
  </body>
  <script type="text/javascript" src="main.js"></script>
</html>

找到ng-app，告诉浏览器开始由ng接管，继续找到ng-model后生成say模型，并挂载在根作用域$rootScope上,在ng-app里都可以通过调用它

视图View：(利用directive实现)

指令(directive)：
var app = angular.module("app", []);
app.directive("kk", function() {
  return {
    restrict: 'E',
    template: '<div> Hi iam kk</div>',
    replace: true
  }
})

<kk></kk>

$scope/$rootScope

ng中mvc都是借助于$scope实现的

$rootScope是根作用域,ng-app里的都可以访问到它

<div ng-controller="GreetCtrl">
  {{name}}
</div>
<div ng-controller="ListCtrl">
  <ol>
    <li ng-repeat="name in names">
      {{name}} from {{department}}
    </li>
  </ol>
</div>

var app = angular.module("app", []);
app.controller("GreetCtrl", ['$scope', '$rootScope', function($scope, $rootScope) {
  $scope.name = 'kk';
  $rootScope.department = 'Greet';
}])
app.controller("ListCtrl", ['$scope', function($scope) {
  $scope.names = ['a', 'b', 'c'];
}])

事件

emit向上传播

broadcast向下传播

<div ng-controller="EventController">
  {{count}}
  <ul>
    <li ng-repeat="i in [1]" ng-controller="EventController">
      <button ng-click="$emit('MyEvent')">1</button>
      <button ng-click="$broadcast('MyEvent')">2</button>
      {{count}}
      <br>
      <ul>
        <li ng-repeat="item in [1,2]" ng-controller="EventController">
          {{count}}
        </li>
      </ul>
    </li>
  </ul>
</div>

var app = angular.module("app", []);
app.controller("EventController", ['$scope', function($scope) {
  $scope.count = 0;
  $scope.$on('MyEvent', function() {
    $scope.count++;
  });
}]);

点击button1触发$emit('MyEvent')，该层和该层以上的count都会自加1

点击button2触发$broadcast('MyEvent')，该层和该层以下的count都会自加1

$scope
  • $scope是一个js对象
  • $scope提供了一些工具方法$watch()/$apply()，内部调用的
  • $scope是表达式的执行环境(即作用域)
  • $scope是一个树型结构，也DOM平行的(即子$scope可以继承父$scope的属性和方法)
  • 没个angular应用都有一个根$scope即$rootScopt(一般位于ng-app上)
  • $scope可以传播事件的 如emit和broadcast
  • $scope不仅是MVC基础，也是双向数据绑定的基础
  • 对$scope调试可以用angular.element($0).scope()来获取当前元素的$scope
