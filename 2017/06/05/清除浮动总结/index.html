清除浮动总结

发表于 2017-06-05 | 分类于 css

清除浮动4种方法（实际上根据原理不同可以分为两类）

第一种：添加一个空的子元素

由于该方法要额外的添加一个空标签，所以通常不采用这种

第二种也是利用父元素，更简单，设置overflow属性
第三种是给父元素也设置float但这会带来其他问题，所以不采用
第四种 以上几种方法都有优缺点，以下这种利用父元素添加伪类实现，比较完美,原理和第一种类似

可以写个通用的样式，直接给父元素添加该类名(clearfix)即可

总结

清除浮动的方法其实是两类，第一类在父元素上触发BFC,第二类在父元素最后添加一个空的元素。

第一类

以下这些属性都可以触发BFC，但是它们清除浮动的效果不一样

能触发BFC的都能清除浮动

第二类

父元素最后添加一个空的元素

这个最优雅，既能清除浮动还能解决margin重叠
