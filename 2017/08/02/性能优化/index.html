性能优化

发表于 2017-08-02 | 分类于 性能优化
性能优化 (没有标准答案，是一个综合性问题)
  • 原则
    • 多使用内存，缓存等方法
    • 减少cpu计算，减少网络
  • 从哪里入手
    • 加载页面和静态资源
    • 页面渲染

  • 加载资源优化

    • 静态资源的压缩合并

    • 静态资源缓存

+ 使用CDN让资源加载更快

+ 使用SSR后端渲染，数据直接输出到HTML中

  • 渲染优化

    • css放前面，js放后面

    • 懒加载 (图片懒加载，下拉加载)

    • 减少DOM查询，对DOM查询做缓存

    • 减少DOM操作，多个操作尽量合并在一起

    • 事件节流

    • 尽早执行操作(如DOMContentLoaded)
