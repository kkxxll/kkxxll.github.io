本地数据库导出到服务器

发表于 2017-08-14 | 分类于 Mongodb

打开本地终端

sudo ./mongodump -h 127.0.0.1:27017 -d mydb -o mydb-backup

把mydb这个数据库导出到mydb-backup这个文件夹下

tar zcvf mydb-backup.tar.gz mydb-backup

接着把文件夹进行打包，打包成mydb-backup.tar.gz

接着把打包后的发给服务器

发送之前，先在服务器下新建一个文件夹用来接收传来的文件

在本地发送文件到服务器

scp -P 服务器端口号 ./mydb-backup.tar.gz username@119.23.43.161:/home/username/之前新建的文件夹名/

到服务器上解开包

tar xvf mydb-backup.tar.gz

服务器导入数据库

mongorestore -d mydb ./dbbackup/mydb-backup/mydb/
