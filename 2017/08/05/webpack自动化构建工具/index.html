webpack自动化构建工具

发表于 2017-08-05 | 分类于 自动化
安装webpack

sudo npm install webpack -g

项目也安装webpack

npm install webpack@1.15.0 –save-dev

编辑webpack.config.js

命令行输入 webpack即可

webpack多个文件输入／输出

综上，webpack既优化了加载速度又能提供一些全局的模块，还免去了每个页面的require

打包css

安装 npm i extract-text-webpack-plugin –save-dev 会报错

提示npm3+版本下，要自己手动去下载该模块

打开github找extract-text-webpack-plugin找历史版本

npm i extract-text-webpack-plugin@1.0.1 –save-dev 会成功

安装webpack-dev-server
