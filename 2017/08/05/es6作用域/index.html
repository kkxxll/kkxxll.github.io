es6作用域

发表于 2017-08-05 | 分类于 es6

作用域

function test() {
  for(let i=1;i<3;i++) {
    console.log(i);
  }
  console.log(i);
}
test();

结果: 1 2 报错

function test() {
  for(var i=1;i<3;i++) {
    console.log(i);
  }
  console.log(i);
}
test();

结果: 1 2 3

let a = 1;
let a = 2;  //报错
var b = 3;
let b = 4; //报错
var c = 5;
var c = 6;

声明过的不能再用let声明

const PI = 3.1415926;
PI = 1; //报错
console.log(PI);

const是存数值时，只读，不能修改

function test() {
  const PI = 3.1415926;
  console.log(PI);
}
test();
const PI = 3.14;
console.log(PI);
const a;        //报错   声明时候必须赋值

不同块作用域下可以定义相同的const

const存对象时候，可添加可修改

const kk = {
  a:1,
  b:2
}
kk.c = 3        //添加
console.log(kk);
kk.a = 11;      //修改
console.log(kk);

结果:
Object {a: 1, b: 2, c: 3}
Object {a: 11, b: 2, c: 3}

总结：const 存对象时候，对象是引用类型的，指向该对象的指针不变，但是对象可以变化。
