less语法

发表于 2017-08-20 | 分类于 less/scss
@charset “utf-8”; 在定义中文时候不会乱码
在less编辑后生成css，在html引用的还是css
导入
less @import "forless"    不需要后缀
css  @import "forcss.css"  要后缀，这样会以@import "forcss.css"出现
     @import (less) "a.css"; 这样会把a.css里的样式嵌入
less中的注释两种：
//我不会被编译
/*我会被编译*/
less中的变量
@变量名: 值

@test_width: 300px;
.box {
    width: @test_width;
}
混合（无参）
body {  
  .color
}
.color {
  color: yellow;
}
等价于
body {
  color: yellow;
}
.color {
  color: yellow;
}
混合(有参)
body {
  .color(red)
}
.color(@c) {
  color: @c;
}
混合(有参 使用默认值)
body {
  .color()  //有值则会覆盖默认值
}
.color(@c:yellow) {
  color: @c;
}
匹配模式
.pos(a) {
  position: absolute;
}
.pos(r) {
  position: relative;
}
body {
  .pos(r)
}
all @_ 所有的都运用该样式
.pos(a) {
  position: absolute;
}
.pos(r) {
  position: relative;
}
.pos(@_) {
  background-color: #777;
}
body {
  .pos(r)
}
运算
@test_color: #777;
@test_width: 300px;
body {
  background: #333+100; //不常用，且不能用变量做运算即@test_color＋100会报错
  width: @test_width+20;
}
嵌套（常用）
body {
  background: #777;
  div { //body div
    background-color: red;
  }
  a {  //body a
    background-color: blue;
    &:hover {   //&表示父级  body a:hover
      background-color: green;
    }
  }
}
@arguments 变量
@arguments包含了所有传递进来的参数
body {
  .border_arg();
  // .border_arg(2px); 也可以传递参数
}

.border_arg(@w:30px,@c:yellow, @xx:solid) {
  border: @arguments;
}
避免编译
css3中有个属性  calc(200px-20px);
该属性是让浏览器取计算
若用less则它会想帮忙计算
所以需要～来避免
less
body {
  width: ~'calc(200px-10px)';
}
css:
body {
  width: calc(200px-10px);
}

less
body {
  width: calc(200px-10px);
}
css:
body {
  width: calc(190px);
}
作用域问题
div {
  @a: red;  //只能在div中用
  color: @a;
}
