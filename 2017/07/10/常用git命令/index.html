常用git命令

发表于 2017-07-10 | 分类于 git
  • git add . 添加
  • git checkout xxx 还原
  • git commit -m “xxx” 放到本地仓库修改
  • git push origin master 上传到远程仓库
  • git pull origin master 从远程仓库下载下来
  • git branch 看当前的分支
  • git checkout -b xxx 新建一个分支
  • git checkout xxx 切换到已有的分支
  • git merge xxx 合并
完整步骤
1.创建一个新的SSH KEY

ssh-keygen -t rsa -C “your_email”

2.在 GITHUB 上添加你的 SSH KEY(Settings–>SSH Keys)
3.验证你的 GITHUB 连接

ssh -T git@github.com

4.配置个人信息

git config –global user.name “your name”

git config –global user.email “your mail”

5.登陆Github网站创建一个新的代码库
6.在本地创建一个文件夹

mkdir testfolder

cd testfolder

7.初始化该文件夹,创建README

git init

vi README

8.添加文件

git add README

添加注释:

git commit -m “first commit”

9.提交文件到Github

若是第一次提交该项目的文件或是修改项目文件名后则需要这行这个命令，以后就可不用执行该命令

git remote add origin https://github.com/yourname/repository_name.git

git push -u origin master
