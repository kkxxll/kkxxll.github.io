配置nginx、域名解析、配置七牛云

发表于 2017-07-29 | 分类于 阿里云
安装nginx

添加CentOS 7 Nginx 到 yum资源库

sudo rpm -Uvh http://nginx.org/packages/centos/7/noarch/RPMS/nginx-release-centos-7-0.el7.ngx.noarch.rpm

使用yum命令从Nginx源服务器中获取来安装Nginx

sudo yum install -y nginx
可通过nginx -v查看版本

配置nginx

cd /etc/nginx/conf.d

sudo vi kk-com-8999.conf

域名解析

给域名kk.cn加上两条A记录

a.kk.cn和b.kk.cn

完成a.kk.cn访问3001端口，b.kk.cn 访问3002端口

七牛云配置

给域名kk.cn加上一条CNAME记录，记录值为七牛云提供的

之后便可以通过自定义的域名访问七牛云上存储的文件
