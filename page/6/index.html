密码安全

发表于 2017-05-19 | 分类于 web安全
密码泄露渠道
  • 数据库被偷
  • 服务器被入侵
  • 通信过程中被窃听
密码安全防御：
  • 不要明文存醋密码
哈希算法：
  • 明文、密文一一对应
  • 雪崩效应 不同密码(即明文)对应的密文可能相同，但是几率很小
  • 密文无法反推出明文
  • 密文的长度是固定的
  • 常见的哈希算法md5、sha1、sha256

Mongodb安装与shell

发表于 2016-12-20 | 分类于 Mongodb
安装与启动:

下载mongodb解压以后

进入bin目录

sudo ./mongod

没有参数的情况下使用默认数据库data/db，如果想要改变可以用 --dbpath 来指定新位置

启动后，在另个窗口输入 mongo 命令，运行shell

文档与集合:

文档是键值对的一个有序集，它是MongoDB的核心概念。

集合是一组文档，如果把文档比喻为数据库中的一行，那么集合就是一张表。

在js中，文档被表示为对象 {“greeting”:”hello world”}
这里，只有一个键”greeting”，其对应的值为”hello world”

键是字符串，命名规则以下:

  1. 键不能含有\0(空字符)，这个字符用于表示键的结尾
  2. 键尽量不要含有.和$

jade Shell

发表于 2016-12-20 | 分类于 模板引擎
  1. 把jade安装到全局下
  2. 得到压缩后的aaa.html

    jade aaa.jade

  3. 若要可读性

    jade -P aaa.jade

  4. 文件变动实时编译

    jade -P -w index.jade

  5. 从命令行中传参数

    jade variable.jade -P -w –obj ‘{“b”:”from cmd”}’
    文档中如果也有b变量的话，文档里的优先级更高

  6. 从json中传递参数

    jade variable.jade -P -w -O variable.json

  7. 没有定义的值为undefined或空

    input(value=’#{nothing}’)

    input(value=nothing)
