浏览器存储

发表于 2017-07-09 | 分类于 本地存储

知识点

  • cookie
  • sessionStorage和localStorage

  • 使用document.cookie = … 获取和修改即可

  • cookie

    • 本身用于客户端和服务器端通信
    • 但是它又本地存储的功能，于是就被借用
  • cookie用于存储的缺点
    • 存储量小，只有4KB
    • 所有http请求都带着，会影响获取资源的效率
    • API简单，需要封装才能用 document.cookie = …
    • 后来h5出了sessionStorage和localStorage可以解决
  • sessionStorage和localStorage
    • HTML5专门为存储儿设计，最大容量5M
    • API简答易用
    • localStorage.setItem(key, value);localStorage.getItem(key)
  • sessionStorage和localStorage区别

密码安全

发表于 2017-05-19 | 分类于 web安全
密码泄露渠道
  • 数据库被偷
  • 服务器被入侵
  • 通信过程中被窃听
密码安全防御：
  • 不要明文存醋密码
哈希算法：
  • 明文、密文一一对应
  • 雪崩效应 不同密码(即明文)对应的密文可能相同，但是几率很小
  • 密文无法反推出明文
  • 密文的长度是固定的
  • 常见的哈希算法md5、sha1、sha256
