css布局

发表于 2016-08-01 | 分类于 css

  1. 固定布局:

    当小于设定的像素时候，会出现丑陋的滚动条

    对于大屏幕设备来说，会有大片空白

  2. 流动布局

    度量单位不再是像素，而是百分比

    单独使用流动布局还不足以保证所有元素在一系列设备上保持良好的
    效果，因为在大屏幕上看起来太宽，在小屏幕上看起来太窄

  3. 弹性布局

    与流动布局相似，只是度量单位是em,1em相当于当前字体的大小

浏览器的视口

发表于 2016-05-19 | 分类于 响应式

视口分为

  • 布局视口（虚拟的）
  • 可视视口（即屏幕大小）
  • 理想视口
pc端视口与移动端端视口的区别

在pc端浏览器上，布局视口与可视视口一样大。

在移动端上，可视视口为屏幕的大小，也就是可视视口决定了用户看到的网站区域。而布局视口与屏幕是没有关联的，它是一个虚拟的视口，比屏幕宽很多，网站的内容是显示在该视口上面。用户可以通过缩放查看网站内容，缩放时，布局视口不会改变的。
