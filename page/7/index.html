em和rem

发表于 2016-10-17 | 分类于 css

em是相对长度单位,相对参照物为父元素的font-size

且em具有继承的特点，若父元素没有font-size,那么会继续向上找祖父元素，若最终还是没有找到，就会使用浏览器的默认值即1em = 16px

rem相对参照物是html

当html没有设置时，使用浏览器默认的即1rem = 16px

鼠标事件

发表于 2016-08-20 | 分类于 js
  • click:鼠标左键，键盘回车键
  • dbclick:双击左键
  • mousedown:按下鼠标任意键
  • mouseenter: 光标从元素外部首次移到元素范围之内时触发，该事件不冒泡,- 且后代元素不会触发
  • mouseleave: 位于元素上的光标移动到元素范围之外触发，该事件不冒泡,且- 后代元素不会触发
  • mousemove: 鼠标指针在元素内部移动时重复地触发
  • mouseout: 移到另一个元素时触发
  • mouseover: 首次移入一个元素之内时触发
  • mouseup: 释放鼠标时触发

鼠标的默认行为取消会影响浏览器的默认行为，还会影响其他事件，因为鼠标事件与其它事件紧密联系

在用一个元素上相继触发mousedown和mouseup才会触发click
触发两次click等于一次dbclick

上个例子中，图片设置float,那么父级高度塌陷，脱离了父级的包裹，它下面的文字

css布局

发表于 2016-08-01 | 分类于 css

  1. 固定布局:

    当小于设定的像素时候，会出现丑陋的滚动条

    对于大屏幕设备来说，会有大片空白

  2. 流动布局

    度量单位不再是像素，而是百分比

    单独使用流动布局还不足以保证所有元素在一系列设备上保持良好的
    效果，因为在大屏幕上看起来太宽，在小屏幕上看起来太窄

  3. 弹性布局

    与流动布局相似，只是度量单位是em,1em相当于当前字体的大小
